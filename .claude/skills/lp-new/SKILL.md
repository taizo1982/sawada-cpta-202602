# LP新規作成スキル

src/ ディレクトリに新しいランディングページを作成します。

**重要**: デザイン作成時は必ず `/frontend-design` スキルを使用してください。

## ヒアリング項目

以下を順番に確認してください：

1. **LPの目的**
   - イベント集客
   - 商品販売
   - リード獲得
   - サービス紹介
   - その他

2. **ターゲット層**
   - 年齢層
   - 性別
   - 職業・属性
   - 悩み・課題

3. **使用する画像**（重要）
   - ヒーロー画像があれば先に確認
   - 人物写真の場合、顔の位置を把握
   - 画像パスを聞く or 画像を見せてもらう

4. **必要なセクション**（デフォルトは全て含む）
   - ヒーロー（必須）
   - 課題・悩み（Pain）
   - 解決策（Solution）
   - お客様の声（Testimonials）
   - よくある質問（FAQ）
   - 料金・オファー
   - CTA（必須）
   - フッター（必須）

5. **カラースキーム**
   - メインカラー
   - アクセントカラー
   - 業種に合った提案も可

6. **参考サイト**（あれば）

7. **コンバージョンアクション**
   - フォーム送信
   - 電話
   - LINE追加
   - 資料請求

## 出力ファイル

```
src/
├── index.html    # HTML構造
├── style.css     # スタイル
├── script.js     # インタラクション
└── images/       # 画像フォルダ（プレースホルダー画像の説明をコメントで記載）
```

## 重要なルール

1. **data-cv属性**: CTAボタンには必ず `data-cv="種類"` を付与
   ```html
   <a href="#form" data-cv="cta-click">申し込む</a>
   <a href="tel:xxx" data-cv="tel">電話する</a>
   <a href="https://line.me/..." data-cv="line">LINE追加</a>
   ```

2. **画像のwidth/height**: CLSを防ぐため必ず指定
   ```html
   <img src="images/hero.jpg" alt="説明" width="1200" height="630">
   ```

3. **セマンティックHTML**: 適切なタグを使用
   - `<header>`, `<main>`, `<section>`, `<footer>`
   - 見出しは `<h1>` から順番に

4. **アクセシビリティ**:
   - 全画像にalt属性
   - タップ領域は44x44px以上
   - コントラスト比を意識

5. **レスポンシブ**: モバイルファースト
   - 375pxで見やすいデザイン
   - 孤立文字（1-2文字の行末）を避ける

6. **画像と顔の配置**: 人物写真を使う場合
   - **必ず先に画像を確認**（Readツールで読み込む）
   - 顔の位置を把握してからコンテンツ配置を決定
   - 顔にテキストやボタンを被せない
   - 配置ルール:
     - 顔が右 → テキスト左配置
     - 顔が左 → テキスト右配置
     - 顔が中央 → テキストは上下に配置、または半透明オーバーレイ

## 作成フロー

1. ヒアリングで要件を確認
2. **画像確認**（人物写真がある場合）
   - Readツールで画像を読み込む
   - 顔の位置を確認（左/中央/右、上/中/下）
   - コンテンツ配置方針を決定
3. `/frontend-design` スキルを呼び出してデザイン作成
   - **下記「日本語LP追加ルール」を必ず渡す**
   - 画像の顔位置情報を渡す
4. src/ に HTML/CSS/JS を出力

## 日本語LP追加ルール（/frontend-design に渡す）

`/frontend-design` 呼び出し時、以下を追加指示として含める：

```
【日本語LP必須ルール】

1. 孤立文字の回避
   - 375px幅で1-2文字だけ行末に残らないようにする
   - 長い単語や文節の途中で改行しない
   - 必要に応じて<br>や&nbsp;で調整

2. 日本語タイポグラフィ
   - 本文の行間: line-height: 1.8〜2.0
   - 文字間: letter-spacing: 0.05em
   - 見出しは太めのウェイト（700以上）
   - 游ゴシック、Noto Sans JP、ヒラギノ角ゴ推奨

3. 画像の顔位置
   - 人物写真の顔にテキストを被せない
   - 顔が右 → コンテンツ左配置
   - 顔が左 → コンテンツ右配置
   - 顔が中央 → 上下配置 or 半透明オーバーレイ

4. モバイル最適化（375px基準）
   - タップ領域: 44x44px以上
   - フォントサイズ: 本文16px以上
   - ボタン: 幅100%または十分な余白

5. 読みやすさ
   - 1行あたり25〜35文字程度
   - 段落間に十分な余白
   - コントラスト比4.5:1以上
```

- `/lp-check` で品質チェック
- `/lp-ads` で広告タグ設定
